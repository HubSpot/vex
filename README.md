##vex 


A modern dialog library which has the following assets:

* Highly configurable.
* Easily stylable. 
* Tiny (`6kb` minified, `2kb` minifed + gzipped).
* Clear and simple API.
* Works on mobile.
* Can be customized to match your style in seconds.
